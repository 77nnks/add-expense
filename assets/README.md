# リッチメニュー画像

`richmenu.png` をこのディレクトリに配置してください。

## 画像仕様

| 項目 | 値 |
|------|-----|
| サイズ | 2500 x 843 px |
| フォーマット | PNG または JPEG |
| ファイルサイズ | 1MB以下 |

## レイアウト

3つのボタンを横並びで配置：

```
┌─────────────┬─────────────┬─────────────┐
│             │             │             │
│   ヘルプ    │    集計     │    更新     │
│     ?       │     📊      │     🔄      │
│             │             │             │
└─────────────┴─────────────┴─────────────┘
   0-833px      833-1667px    1667-2500px
```

## 作成方法

1. **Canva** (https://www.canva.com)
   - カスタムサイズ 2500 x 843 px で作成

2. **Figma** (https://www.figma.com)
   - フレームサイズ 2500 x 843 px で作成

3. **簡易画像生成サイト**
   - https://line-richmenu-maker.netlify.app/ など

## 配色例

- 背景: #4A90A4（ブルーグレー）
- ボタン: #FFFFFF（白）
- テキスト: #333333（ダークグレー）
